ℹ️ Como configurar o agendamento automático (Cron) ▼

O campo de agendamento usa um formato chamado cron, que permite definir exatamente quando uma tarefa deve ser executada automaticamente. Ele é composto por 5 campos separados por espaço, cada um representando uma unidade de tempo.

A ordem dos campos é: Minuto — Hora — Dia do mês — Mês — Dia da semana

Exemplos prontos para usar:

Expressão Quando executa
* * * * * A cada minuto Usar ↗
0 8 * * * Todo dia às 08:00 Usar ↗
0 18 * * 1-5 De segunda a sexta às 18:00 Usar ↗
*/5 * * * * A cada 5 minutos Usar ↗
0 0 1 * * Dia 1 de cada mês à meia-noite Usar ↗

Dicas rápidas:

  • O símbolo * significa "qualquer valor" — todos os minutos, todas as horas, todos os dias, etc.
  • Use / para intervalos — */10 significa "a cada 10".
  • Use - para faixas — 1-5 significa "de 1 até 5".
  • Use , para valores específicos — 1,15,30 significa "nos valores 1, 15 e 30".

⚠️ Os horários seguem o fuso horário configurado no sistema. Certifique-se de que ele está correto antes de salvar o agendamento.
